# HF_Calculator_WPF

A simple calculator application built with WPF (Windows Presentation Foundation) using C# and XAML.  
This project was created for learning purposes to practice event handling, UI layout, and basic arithmetic logic.

## Features
- Number buttons (0–9)
- Basic operations: addition, subtraction, multiplication, division
- Percentage calculation
- Clear and readable display
- Minimalistic, dark-themed UI

## Technologies Used
- C#
- .NET / WPF
- XAML
- Visual Studio 2026

## Current Status
The project is still in an early stage, but fully functional.  
Future improvements may include:
- A grid-based layout that scales properly when resizing the window
- More advanced operations
- Better error handling and input validation
- A more responsive UI

## How to Run
1. Clone the repository  
2. Open the solution in Visual Studio 2026
3. Build and run the project
